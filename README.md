# Southeast Asia GDP Growth & Forecast (1960–2024)

## Overview
Analysis of economic growth patterns across 11 Southeast Asian countries 
over 64 years, using machine learning to cluster economies and forecast 
GDP trends through 2030.

## Key Results
- Applied PCA + K-means clustering with 98.76% explained variance
- Identified 3 distinct economic tiers across Southeast Asia
- Forecasted Vietnam's GDP to reach $740.40B by 2030, surpassing 
  Thailand as the 3rd largest economy in Southeast Asia

## Tools & Technologies
- Python (Pandas, Numpy, Matplotlib, Seaborn, Sklearn)
- Machine Learning: PCA, K-means Clustering, Log-Linear Regression
- Tableau (Dashboard)
